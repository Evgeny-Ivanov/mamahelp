Welcome to MamaHelp

TThis app is part of a prototype series of BeeHelper project, that serves the purpose to experiment with flows for managing of multi-steps forms and authentication. For demo purposes there exist an oversimplified backend part that processes authentication requests and data storage/retrieval. Source code for this project is available on my GitHub. Feel free to explore.

Available features

Authentication

We have a simple authentication form here, where user can enter credentials or proceed to registration form in case if credentials don’t exist. Simple HTML form validation is present for both authentication and registration forms that operate on user inputs. For concise purposes validation that requires backend, interaction is not present here.

To explore, how the rest of the application works you can use already existing credentials:

or register a new user. TThis app doesn’t perform any email verification so feel free to enter any valid fake email address.

Need help and can help

These features are available for the authenticated user. We can see a list of previously filled requests as well as create a new one. Need help are system records that represent your requests for getting some help and is filled through a multistep form. Can help form is an opposite one and is going to serve a purpose of showing a willingness to provide some services.

Feel free to create/update/delete records here, but leave at least one record for others.
